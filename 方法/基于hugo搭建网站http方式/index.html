基于hugo搭建网站（http方式）

材料：1.安装hugo，2.安装git,3.安装编写markdown语言编译器（VS studio），4.GitHub账号、主题

1.下载hugo、git、VS，安装后配置路径

2.打开cmd，输入hugo，检查是否正确配置路径，若正确，输入hugo new site yourname.com 此时，在电脑c盘的用户路径上出现yourname.com文件夹

3.下载主题（ https://themes.gohugo.io/ ）解压在yourname。com文件夹内

4.配置主题（不同主题配置方式不一，但主要是配置config.toml文件），以改网站的hermit为例，将example site中的content、config.toml替换yourname目录下的相应文件

5.VS studio打开文件—打开终端—输入hugo server(本地打开)

6.VS中打开config文件，修改内容（baseurl=“https://yourname.com”）

7.github创建库githubname\yourname.github.io

8.yourname目录下gir bush here输入hugo，生成public文件夹

9.键入后，继续git bush 依次输入git init（若为empty则在输入一次） git add -A git commit -am"init" 如果为首次使用，需先登陆（git remote add origin https://github.com/githubname/yourname.github.io.git弹出框登陆github即可） git push -f origin master

常见问题

1.登入新账号 此前我已使用hoxx搭建 https://jontycleopoly.github.io/，因此每次登陆时，会自动登上这个已不使用的网站，这时需打开控制面板—外观和个性化—用户账户—管理Windows凭据—GitHub下拉删除账户信息即可

2.github无法正常加载主题 GitHub仓库两分支，改名main—master（需要删除原来的库再新建后重命名，重新git）

3.更新 VS使用MD编辑，保存，可先使用hugo server观看本地效果 上传github hugo git add -A git commit -m"xxx" git push -f origin master
